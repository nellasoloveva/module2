### 1.1. Топология сети (Схема)

```ascii
      +----------+
      | Internet |
      +----------+
           |
           | [⚠️net0⚠️/⚠️net6⚠️]
        +-----+ [⚠️net2⚠️/⚠️net8⚠️]
        | ⚠️ISP⚠️ |-----------------+
        +-----+                 |
           | [⚠️net1⚠️/⚠️net7⚠️]        |
           |                    |
        (⚠️ISP⚠️-HQ)             (⚠️ISP⚠️-BR)
           |                    |
           | [⚠️net1⚠️/⚠️net6⚠️]        | [⚠️net1⚠️/⚠️net6⚠️]
       +--------+           +--------+
       | ⚠️HQ-RTR⚠️ |=== GRE ===| ⚠️BR-RTR⚠️ |
       +--------+           +--------+
   (Trunk) | [⚠️net2⚠️/⚠️net7⚠️]        | [⚠️net2⚠️/⚠️net7⚠️]
           |                    |
    +------+------+             |
    |             |             |
(SRV-Net)     (CLI-Net)      (BR-Net)
    |             |             |
    | [⚠️net1⚠️/⚠️net6⚠️] | [⚠️net1⚠️/⚠️net6⚠️] | [⚠️net1⚠️/⚠️net6⚠️]

| ⚠️HQ-SRV⚠️ |    | ⚠️HQ-CLI⚠️ |    | ⚠️BR-SRV⚠️ |
+--------+    +--------+    +--------+
```
